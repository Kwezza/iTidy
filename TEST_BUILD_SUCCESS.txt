Purpose:
--------
Benchmark the ListView API performance on 7MHz Amiga
to determine practical limits before UI becomes sluggish.

Expected Performance:
- 50 rows: < 50ms format + sort
- 100 rows: < 100ms operations
- 200 rows: < 300ms sort
- 500+ rows: Too slow for 7MHz

Test by repeatedly adding 50 rows and observing
when sort operations become noticeably laggy.
