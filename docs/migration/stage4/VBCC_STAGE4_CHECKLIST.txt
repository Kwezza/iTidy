Known Issues:
-------------
1. FontPrefs structure incomplete type warning
   Status: Does not affect compilation or runtime
   Impact: None

2. Workbench 2.x icon spacing wider (15x10 vs 9x7)
   Status: By design - limited icon.library support
   Impact: Visual only, functionally correct

3. NewIcons not visible on WB 2.x without NewIcons
   Status: Use -forceStandardIcons flag
   Impact: Data loss if not using flag
