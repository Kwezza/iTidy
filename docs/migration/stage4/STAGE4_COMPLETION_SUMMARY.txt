TROUBLESHOOTING
===============================================================================

"Library not found" errors:
  - Verify Workbench 2.0 or higher
  - Check icon.library, dos.library available
  - Try CLI first (less complex than Workbench launch)

"Device is read-only" error:
  - Expected for CD-ROM, write-protected disks
  - Test with writable device (DH0:, Work:, etc.)

No log file created:
  - Check write permissions
  - Verify Bin/Amiga/logs/ directory exists
  - Should fall back to T:iTidy.log

Crashes or hangs:
  - Note exact command run
  - Check if specific directory causes issue
  - Try smaller test directory first
  - Provide log file up to crash point
